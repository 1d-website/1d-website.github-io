How to Use

Admin Login

To access the admin panel, click on the "Admin Login" button on the homepage. Enter the passcode. And you will be sent to the admin panel.

Admin Panel

In the admin panel, you can add homework by filling out the form with the due date, subject, and homework details. Only authorized phone numbers can add homework.

You can also view and delete existing homework from the list.

Homework List

On the homepage, you can view the homework list by selecting the year and month, then clicking the "Generate Calendar" button. Click on a date to view the homework details for that day.

Homework Details

On the homework details page, you can view the homework assigned for the selected date.
